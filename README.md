## Project Overview

This repository contains the complete progression of cybersecurity internship tasks from Week 1 to Week 6. The project starts with a vulnerable Node.js/SQLite web application, then moves through vulnerability remediation, advanced API security, ethical hacking, exploitation testing, CSRF protection, final security audits, Docker deployment, and secure deployment practices.

The main goal of this project is to demonstrate practical web application security skills, including vulnerability identification, exploitation, remediation, secure coding, auditing, and deployment hardening.
